XEWeb are proud to announce the release of Flintstone, a key/value database store using flat files for PHP.

Some of the features include:

  • Memory efficient

  • File locking

  • Caching

  • Gzip compression

  • Easy to use

All it requires is PHP 5 and read/write file permissions.
