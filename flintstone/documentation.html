Flintstone::load(string $database, array $options)

Load a database with options, returns an object to be used for the methods below:

  • $dir - the directory where the database files are stored (this should be somewhere that is not web accessible)
    e.g. /path/to/database/
  • $ext (.dat) - the database file extension to use
    e.g. .txt
  • $gzip (false) - use gzip to compress the database
    e.g. true
  • $cache (true) - store get() results in memory
    e.g. false
  • $formatter (object) - the formatter class used to encode/decode data
    e.g. new JsonFormatter()
  • $swap_memory_limit (1048576) - amount of memory to use before writing to a temporary file
    e.g. 0

get(string $key)

Retrieve data for the key name. Returns false if it does not exist.

set(string $key, mixed $data)

Set data for the key name. Data can be a string, integer, float or array. Returns true or will throw an exception if fails to set.

replace(string $key, mixed $data)

Replace data for the key name. Using set() on an existing key has the same effect. Returns true or will throw an exception if fails to set.

delete(string $key)

Delete the key name. Returns true or will throw an exception if fails to delete.

flush()

Empty the database. Returns true or will throw an exception if fails to flush.

getKeys()

Returns an array of all of the keys in the database.

getAll()

Returns all data in the database.

Flintstone::unload(string $database)

Unload a database (destroys the object)
