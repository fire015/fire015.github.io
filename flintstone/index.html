A key/value database store using flat files for PHP.

Features include:

  • Memory efficient
  • File locking
  • Caching
  • Gzip compression
  • Easy to use

Installation

To install Flintstone run the following command with Composer:

{% highlight bash %} composer require fire015/flintstone {% endhighlight %}

Or add to your composer.json file:

{% highlight json %} { "require": { "fire015/flintstone": "2.*" } } {% endhighlight %}

PHP Example:

{% highlight php %} '/path/to/database/dir/']; $users = new Flintstone('users', $options); {% endhighlight %}

Requirements

  • PHP 5.6+

Data types

Flintstone can store the following data types:

  • Strings
  • Integers
  • Floats
  • Arrays
  • Null
